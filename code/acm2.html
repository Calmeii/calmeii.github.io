函数的使用准则

函数是什么?

  • 函数是一段指令集合
  • 可以在任意一行程序调用函数指令
  • 函数的输入：参数
  • 函数的输出：返回值、参数、异常

注意：在竞赛中，参数与返回值尽量避免使用指针

函数的基本应用

  • 输入一个字符串,逆向输出该串

递归的思想

经典汉诺塔问题

Hanoi是世界末日问题
相传当Hanoi问题被解决时，世界末日就来临了
使用计算机找找规律，解决它汉诺塔

问题: ABC代表左中右三个杆子,将A杆子上n个盘子借助B杆子移动到C杆子上,把将A杆子最顶层的盘子移动到B上的移动方案，记作A>B。我们需要将A杆上的N个盘子通过B杆全部转移到C杆上。按顺序输出移动盘子的方案。

规律:

  1. 将X上n-1个盘子借助Z移到Y上
  2. 将X最下面一个盘子移到Y(输出)
  3. 将Y上n-1个盘子借助X移到Z上

  • 找到一个“通用”的解决问题的方法！
    设计一个函数功能为a柱借助b柱将n个盘子移动到c柱
    void hanno(char a, char b, char c, int n);

  • 这个问题是可以变小的！！
    每一个hanoi(n)都会调用hanoi(n-1);
    所以n的值会不断减小,直到n=1时就不用再调用函数, 直接输出.

      #include<cstdio>
  void hanno(char a, char b, char c, int n)
  {
      if(n == 1){
          printf("%c>%c\n", a, c);
          return;
      }
      hanno(a, c, b, n-1);
      printf("%c>%c\n", a, c);
      hanno(b, a, c, n-1);
  }

  int main()
  {
      int n;
      cin >> n;
      hanno('A', 'B', 'C', n);
      return 0;
  }

深度优先搜索

迷宫

  • 给你一个迷宫,求解是否能够逃出迷宫


一种简单的思想:对于每一个点,按照一定规律向搜素(比如左右上下),一直走到出口,或者走到没有路再走的情况,这时候,返回到上一步,进行下一步搜索.
对于这一一个迷宫搜索路径为 1->5->9->8->4,无路可走回到8->12,无路可走回到8,回到9->.....15到达出口.

bool flag = false;

dfs(x,y)
{
    如果x,y是出口,标记flag = true;
    如果flag = true, return; 已经找到结果便不在搜索
    对于能走到的所有坐标 nextx,nexty, dfs(nextx,nexty);
}

  • 如何做好路径并标记?
    可以在搜索的时候加上一个深度n,用一个数组存下每一个深度n的坐标.

          x[100], y[100];
      dfs(int x, int y, int n);

  • 如果我们已经知道出口离我们最多有100米,如何求出最短路径?

    • 进行深搜的时候如果深度已经大于100 便不再继续

    • 记录当前点到出口的最短路长度

四皇后问题

在棋盘上放置四个皇后,使他们互不攻击,此时每个皇后的攻击范围为同行同列和同对角线,要求找出所有解.

解决办法:从最左列到最右列依次遍历所有能放置的位置,

每一列都具有相同的问题：

1. 当前列所有格子都需要进行尝试
2. 当前格子是否能放皇后
3. 放置皇后对后续产生的影响
4. 如何进行二次尝试
5. 如何避免重复尝试
6.
