线性表

线性表（亦作顺序表）是最基本、最简单、也是最常用的一种数据结构。线性表中数据元素之间的关系是一对一的关系，即除了第一个和最后一个数据元素之外，其它数据元素都是首尾相接的。线性表的逻辑结构简单，便于实现和操作。因此，线性表这种数据结构在实际应用中是广泛采用的一种数据结构。
线性表是使用线性逻辑存储的数据结构

以下两种数据结构不是线性表结构

栈

栈（stack）又名堆栈，它是一种运算受限的线性表。其限制是仅允许在表的一端进行插入和删除运算。这一端被称为栈顶，相对地，把另一端称为栈底。向一个栈插入新元素又称作进栈、入栈或压栈，它是把新元素放到栈顶元素的上面，使之成为新的栈顶元素；从一个栈删除元素又称作出栈或退栈，它是把栈顶元素删除掉，使其相邻的元素成为新的栈顶元素。
先进后出

type stack[N];
int top = 0;
void push (type t ) {    //向栈中压入 t.
    stack[top++] = t;
}

type pop ( ) {
    if  (top > 0 ) return stack[--top]; //弹出栈顶, 并返回新栈顶.
}

stack[top-1](top>0) 读取栈顶.

看一个站的经典题目 Rails
火车车厢需要重新排列，利用车站是否能调整出期望的顺序
本题有两种操作

1. A最左边车厢进入Station,到达Station最上部.
2. Station 最上部车厢,离开Station到达B.

不难看出Station 是一个栈,先进的车厢后出,后进的先出.

进站顺序：1 2 3 4 5
出站顺序：
1 2 3 4 5 -> Y
5 4 3 2 1 -> Y
5 4 1 2 3 -> N
思路 :
定义一个栈来模拟Station.
从左往右扫描出站顺序,若栈顶车厢和当前扫描到的车厢不同,则向栈中压入A最左车厢.
若A中车厢已经全部压入栈中,而B还没有完成扫描则不可能完成相应的出站顺序.

#include <cstdio>
const int maxn = 1010;
int Stack[maxn];
int B[maxn];
int top = 0;

int pop()
{
    if(top > 0) return Stack[--top];
}

void push(int t)
{
    Stack[top++] = t;
}

int main()
{
    int n;
    while(scanf("%d", &n), n){
        while(scanf("%d", &B[0]), B[0]){    //读入每一行第一个数,读入数为0则跳出.
            top = 0;
            for(int i = 1; i < n; i++){
                scanf("%d", &B[i]);
            }
            int pb = 0;     //当前扫描的车厢
            int pa = 1;     //当前A最左边车厢
            while(pb < n){  //还没有扫瞄完B就继续
                if(top>0 && Stack[top-1] == B[pb]){ //如果栈顶和当前扫描车厢相同
                    pop();  //栈顶出栈
                    pb++;   //向右扫瞄
                }
                else{
                    if(pa > n)break;    //如果前n个车厢已经压入且不符合要求,需要压入第n+1个车厢,则退出循环
                    push(pa++);     //压入A最左车厢
                }
            }
            if(pb == n)printf("Yes\n");
            else printf("No\n");
        }
        printf("\n");
    }
    return 0;
}

括号匹配问题
( ) ( ( ( ( ) ) ) ( ) ( ) ( ) ) ( ( ) ( ) ( ) ) ) ) ) ( )是否合法匹配
从左往右扫,左括号压栈,右括号则弹出栈顶的左括号,若栈为空栈或者栈顶不是左括号,则不合法.若扫描完整后栈中还有括号则不合法 反之合法.

复杂括号匹配问题
([{}{}()(([)])))])是否合法匹配
和上题类似,判断时注意括号形状匹配

表达式求值
求出((3+7)(28)-5)/10+50的值
首先
1.表达式肯定合法
2.乘法 和除法的优先级高于加减法.
3.括号内的一定优先计算.

解决:两个栈实现, 一个栈保存括号和值,一个栈保存运算符号,若下一个符号括号,则弹出值栈里面的值进行计算,直到弹出左括号为止,再将计算结果压入值栈,若扫描到乘除号则值栈取顶,并扫描下一个数值,计算结果压入栈中.
扫描完成后 栈中就只涉及加减法的运算, 弹出运算即可.

队列


队列是一种特殊的线性表，特殊之处在于它只允许在表的前端（front）进行删除操作，而在表的后端（rear）进行插入操作，和栈一样，队列是一种操作受限制的线性表。进行插入操作的端称为队尾，进行删除操作的端称为队头。

type q[N];
int head = 0, tail = 0;

void enqueue(type t) {
    q[tail++] = t; //向队尾添加成员
}

void dequeue(type t) {
    if (head < tail) return q[head++];}//队首出列一个成员

回到迷宫问题

搜索
搜索可以看作是在一个树形的结构上进行操作
每一次函数调用作为树形结构上的一条边
递归的最终会停止

深搜(dfs)思想：
一直往深处走，直到找到解或者走不下去为

广搜(bfd)思想:
1.从初始状态S开始，利用规则，生成下一层的状态。(下一层加入队列)
2.顺序检查下一层的所有状态，看是否出现目标状态G。(队列取front)
否则就对该层所有状态节点，分别利用规则。生成再下一层的所有状态节点。(下一层再加入队列)
3.继续按上面思想生成再下一层的所有状态节点，这样一层一层往下展开。直到出现目标状态为止。

广度优先搜索的特点
离原始操作更近一些的搜索操作会优先执行
通常情况下使用队列，不需要使用递归

经典应用：查找到迷宫出口的最短路
如下面这道题
走迷宫1

#include <cstdio>
#include <iostream>
#include <stack>
#include <cstring>
using namespace std;
const int maxn = 110;


struct Point
{
    int x, y;//坐标
    int d;  //原点到该点的最小距离d
}line[maxn*maxn];

int m, n;   //地图长宽
int head, tail; //队列首尾
char mp[maxn][maxn];    //保存地图
bool vis[maxn][maxn];   //标记是否访问过
int ans;        //答案
bool flag;      //标记是否找到答案
int dir[4][2] = {-1, 0, 1, 0, 0, -1, 0, 1};

void init() //初始化
{
    head = tail = 0;
    memset(vis,false, sizeof(vis));
}


bool judge(int x, int y)
{
    if(x<0 || x >= n || y < 0 || y >= m)return false;//超出边界
    if(mp[x][y] == '*') return false;   //围墙
    if(vis[x][y] == true) return false; //已经访问过
    return true;
}

void bfs(Point now)
{
    int xx = now.x;
    int yy = now.y;
    if(mp[xx][yy] == 'T'){//当前点是出口
        ans = now.d;
        flag = true;
        return;
    }
    for(int i = 0; i < 4; i++){
        int dx = dir[i][0];
        int dy = dir[i][1];
        if(judge(xx+dx,yy+dy)){//如果下一个点可以走并且没有被访问过则将这个点加入队尾
            Point s;
            s.x = xx+dx;
            s.y = yy+dy;
            s.d = now.d+1;
            line[tail++] = s;
            vis[xx+dx][yy+dy] = true;
        }
    }
}

int main()
{
    scanf("%d%d", &n, &m);
    for(int i = 0; i < n; i++) scanf("%s", mp[i]);
    for(int i = 0; i < n; i++){
        for(int j = 0; j < m; j++){
            if(mp[i][j] == 'S'){//找到奇点并加入队尾
                Point s;
                s.x = i;
                s.y = j;
                s.d = 0;
                line[tail++] = s;
                break;
            }
        }
        if(tail)break;
    }
    flag = false;
    while(head<tail && !flag){//没有找到出口并且
        bfs(line[head++]);
    }
    if(flag) printf("%d\n", ans);
    else printf("-1\n");
    return 0;
}
